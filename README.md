# RAGLab
RAG and Context Architectures

Mapping the design space of Retrieval-Augmented Generation architectures.
This repository is a collection of practical **RAG (Retrieval-Augmented Generation) system architectures** used to build reliable, scalable, and explainable AI applications.
As LLM systems move from demos to production, the challenge shifts from prompting models to **designing robust information pipelines**. Retrieval-Augmented Generation provides a way to ground large language models in external knowledge sources, improving accuracy, transparency, and controllability.

This project explores different **RAG architectural patterns**, including:

- Basic retrieval pipelines
- Multi-stage retrieval
- Query rewriting and expansion
- Reranking architectures
- Hybrid search (vector + keyword)
- Agent-assisted retrieval
- Memory and context strategies
- Evaluation and feedback loops

Each architecture includes:
- conceptual explanation
- system diagrams
- implementation examples
- trade-offs and design considerations

The goal of this repository is to serve as a **reference for engineers, researchers, and builders designing real-world LLM systems.**
